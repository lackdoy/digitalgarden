### Topic : django render() method 역할
---
### Summary
> render()함수를 이용해서 templat을 그려낼수 있다.

#### Content :
- render()함수는 보통 return 해줄때 request 인자와 같이 넘긴다.
- render()함수는 3가지 매개변수를 받는다.
- 첫 번째는 request 객체를, 두 번째는 탬플릿의 파일 명(경로포함), 셋째로 딕셔너리의 경로(어떤 값을 html에 넘겨줄 건지)
